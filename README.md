# Social Development Events Platform

A community-driven platform where users can create, join, and track social service events in their local area.

## Live Site
https://social-development-10.netlify.app/

## Features
- User authentication with Email/Password and Google login.
- Create, manage, and join social service events (Cleanup, Plantation, Donation, etc.).
- Upcoming events page with filter and search functionality.
- Private pages for managing and viewing joined events.
- Responsive design for mobile, tablet, and desktop devices.
- Light/Dark theme toggle.
- Beautiful UI animations using Framer Motion.
- Newsletter and gallery sections on the homepage.

## Tech Stack
- Frontend: React, Tailwind CSS, Framer Motion, React Datepicker, Daisy Ui
- Backend: Node.js, Express, MongoDB
- Hosting: Client (Netlify), Server (Vercel)
